# Тэнүүн - Хамт байвал болох уу?

Наддаа чи минь нар, сар, тэнгэр гэж   
Би худлаа хэлж чадахгүй нь уучлаарай   
Чамгүйгээр байх хэцүү   
Чамайг мартаж чадахгүй болсноо би зүрхэндээ   
Мэдэрч байна хонгор минь    
Чи миний зүрхний тал тийм болохоор    
Би зүрхээ хуурч чадахгүй нь бүсгүй минь   
Үнэнээ л хэлье ээ    

Дх: Аз жаргалтай мөчүүд амьдралд олон байж     
Гуниг зовлон таарах үед гараас чинь хөтлөөд    
Хамтдаа байвал болох уу?     
Хариулаач дээ надад хамтдаа байвал болох уу?     
Жаргал зовлонг хамт туулж ганцхан чамтайгаа     
Амьдармаар байна болох уу? Итгээрэй      

Тансаг оргилуун дарс, бал бурам     
Алт мөнгөөр би баян биш уучлаарай     
Гэвч хүнээс дутахгүй сайхан амьдрал руу тэмүүлэх     
Тэнүүн хүсэл минь бий     
Урмаар тэтгээрэй      
Амьдралд гэрэл байдгийн адил    
Бас сүүдэр бий     
Зүрхэнд минь зул болон гэрэлтүүлэн гийгүүлээрэй      

Дх:     
Насан туршдаа мөрөөр чинь зөөлхөн тэврээд     
Цэлмэг тэнгэрийн од, сар луу очлоо     
Хамт байвал болох уу?     

Наддаа итгээрэй
